## Organização do Projeto 🗂️

Para gerenciar o projeto, utilizamos a seguinte estrutura:

- **Branches**: 
  - **Branch Principal**: `main` 
  - **Branch**: `ExerciciosPython_Squad_Jaqueline_Goes` - Para desenvolvimento dos exercicios propostos.

- **Estrutura de Pastas**:
